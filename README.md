# 얕넓 v0.1

얕고 넓은 지식을 위한 모바일앱 MVP입니다.

## 실행 방법

먼저 Node.js LTS를 설치한 뒤, VSCode에서 이 폴더를 열고 터미널에서 실행하세요.

```powershell
npm install
npm run start
```

Expo 개발 서버가 뜨면 휴대폰의 Expo Go 앱으로 QR 코드를 스캔해 확인할 수 있습니다.

## 현재 구현된 기능

- 홈 탭: 오늘의 지식 3개
- 탐색 탭: 카테고리/전체 지식 목록
- 복습 탭: 더미 복습 목록
- 마이 탭: 더미 학습 통계
- 지식 상세: 카드 5장 넘기기
- 퀴즈: 선택지 선택 후 정답/오답/해설 표시
- 북마크: 화면 상태 기준 토글

## 아직 연결하지 않은 것

- Supabase DB
- 로그인
- 실제 북마크 저장
- 실제 학습 기록 저장
- 푸시 알림
- 결제/구독
